In Informatik und diskreter Mathematik, ein inversion ein Paar von Stellen in einer Folge, bei dem die Elemente an diesen Stellen nicht in ihrer natürlichen Reihenfolge sind. Wenn wir also eine aufsteigende Reihenfolge für eine Gruppe von Zahlen verwenden, dann ist eine Inversion, wenn größere Zahlen vor niedrigeren Zahlen in einer folge.

Sehen Sie sich diese Beispielfolge an: (1, 2, 5, 3, 4, 7, 6) und wir können hier drei Invertierungen sehen
- 5 und 3;
- 5 und 4;
- 7 und 6.

Sie erhalten eine Folge von eindeutigen Zahlen und sollen die Anzahl der Umkehrungen in dieser Folge zählen.

Eingabe: Eine Sequenz als Tupel von ganzen Zahlen.

Ausgabe: Die Inversionsnummer als ganze Zahl.

Beispiel: 

{% if interpreter.slug == "js-node" %}
countInversion([1, 2, 5, 3, 4, 7, 6]) == 3
countInversion([0, 1, 2, 3]) == 0
{% else %}
count_inversion((1, 2, 5, 3, 4, 7, 6)) == 3
count_inversion((0, 1, 2, 3)) == 0
{% endif %}

Wie sie verwendet wird: In dieser Mission werden Sie das Wunder der verschachtelten Schleifen erleben, das heißt natürlich, wenn Sie keine fortgeschrittenen Algorithmen verwenden.

Vorbedingung: 2 < len(Folge) < 200
len(Folge) == len(set(Folge))
all(-100 < x < 100 für x in Folge)
